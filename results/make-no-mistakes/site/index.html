Benchmark selection (Codex)

The skill is a generic prompt-level accuracy/caution directive, and its likely effects on coding are most visible on deterministic tasks where small reasoning, implementation, or repair mistakes flip pass/fail outcomes. I prioritized code repair, competitive programming, and realistic SWE tasks with executable validation over factual QA or broad multimodal reasoning.

  • humanevalfix (fit 0.92) (recommended): Python code repair tasks are compact, deterministic, and highly sensitive to off-by-one errors, missed edge cases, and overconfident edits, so an added caution directive could measurably help or slow/overcomplicate fixes.
  • swebench-verified (fit 0.9) (recommended): Real bug-fix tasks with human-validated issues and test-based grading expose whether extra caution improves repository navigation and patch precision or causes excessive conservatism and incomplete fixes.
  • livecodebench (fit 0.86) (recommended): Algorithmic coding tasks have unambiguous pass/fail tests and are error-prone in reasoning, implementation details, and edge cases, making them a strong probe for a 'be careful' prompt intervention.
  • quixbugs (fit 0.8): Small single-defect repair tasks in Python and Java are deterministic and mistake-sensitive, though many are simpler than full SWE tasks, so the effect may be easier but less representative.
  • bigcodebench-hard-complete (fit 0.78): Challenging Python programming tasks with reward-based verification should reveal changes in precision and edge-case handling, especially if the skill affects solution planning or test awareness.
  • aider-polyglot (fit 0.74): Polyglot editing tasks broaden coverage across languages and editing patterns, useful for detecting regressions from verbosity or hesitation, though effects may be noisier across heterogeneous tasks.
  • swe-gen-js (fit 0.72): Large JavaScript/TypeScript bug-fix set with realistic issues can reveal coding-agent pass-rate shifts, but generated task provenance makes it slightly less clean than verified real-world sets.
